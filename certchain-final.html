Comment ça marche

Trois étapes simples pour une certification permanente et vérifiable.

01

L'université émet

L'établissement autorisé remplit le formulaire. Le système génère un hash SHA-256 unique du diplôme.

02

Ancrage blockchain

Le hash est enregistré dans le smart contract Ethereum via une transaction signée. Immuable, tracé, horodaté.

03

Vérification libre

N'importe qui — recruteur, institution — interroge le contrat avec le hash ou le QR code. Résultat instantané.

04

Audit permanent

Chaque événement est loggé on-chain. L'explorateur montre l'historique complet de toutes les certifications.
